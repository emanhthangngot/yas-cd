\subsection{Hướng Dẫn Chung Chạy Test}

Do project sử dụng cấu trúc monorepo với thuộc tính \texttt{\$\{revision\}}, lệnh phải chạy từ bên trong thư mục module tương ứng:

\begin{lstlisting}[language=sh, caption={Lệnh chạy test và tạo report JaCoCo}]
cd /path/to/yas/<module>
./mvnw -f ../pom.xml test -pl <module> -am
./mvnw -f ../pom.xml test jacoco:report -pl <module> -am
open target/site/jacoco/index.html
\end{lstlisting}
